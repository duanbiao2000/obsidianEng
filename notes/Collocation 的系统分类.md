**底层需求：** 理解 collocation 的系统分类，便于有针对性地学习和使用。

---

按**结构关系**分，主要六类：

**1. 动词 + 名词** make a decision / take a risk / pay attention 最高频，也最容易用错（中文思维会说 _do a decision_）

**2. 形容词 + 名词** heavy traffic / strong coffee / deep sleep 注意：heavy/strong/deep 在中文里都是"重/强/深"，但搭配完全不同

**3. 名词 + 名词** time management / traffic jam / salary cut 复合名词型，语感靠积累

**4. 副词 + 形容词** deeply committed / highly recommended / utterly exhausted 副词不能乱换，_very committed_ 可以，_strongly committed_ 就弱很多

**5. 动词 + 副词** apologize sincerely / sleep soundly / think clearly 副词修饰动词，强化语义精度

**6. 名词 + 动词** the economy grows / rumors spread / tension rises 主语选什么动词，有固定倾向

---

**另一种分法（按紧密程度）：**

|类型|说明|例子|
|---|---|---|
|Free collocation|可灵活组合|eat quickly|
|Restricted collocation|有倾向但不唯一|make/take a decision|
|Idiom|固定，不可拆|kick the bucket|

---

**实际学习建议：** 优先攻克 **动词+名词** 和 **形容词+名词** 这两类，覆盖率最高，错误率也最高。


## 动词 + 名词（最重要）

**make** make a decision / mistake / effort / progress / difference / money / sense / a point

**take** take a risk / break / chance / responsibility / action / notes / place / time

**do** do damage / harm / research / business / a favor / the dishes / your best

**have** have a meeting / conversation / impact / doubt / potential / influence / experience

**get** get permission / access / attention / results / feedback / rid of / involved

**pay** pay attention / a visit / a compliment / the price / respect

**give** give advice / a speech / permission / support / a hand / feedback

---

## 形容词 + 名词（坑最多）

|中文直觉|错误用法|正确搭配|
|---|---|---|
|强烈的茶|~~strong tea~~|strong tea ✓（这个对）|
|强烈的风|~~strong wind~~|strong wind ✓|
|沉重的交通|~~heavy traffic~~|heavy traffic ✓|
|沉重的雨|~~heavy rain~~|heavy rain ✓|
|深度睡眠|~~deep sleep~~|deep sleep ✓|
|深度问题|~~deep problem~~|serious problem ✓|
|高度重要|~~high importance~~|great importance ✓|

**high** → high risk / pressure / quality / price / speed **strong** → strong coffee / wind / accent / evidence / feelings **heavy** → heavy rain / traffic / smoker / workload / price **deep** → deep sleep / breath / understanding / roots / trouble **great** → great deal / importance / success / impact / pleasure **serious** → serious problem / injury / concern / mistake / threat

---

## 副词 + 形容词

deeply committed / concerned / troubled / rooted highly recommended / competitive / skilled / unlikely utterly exhausted / confused / ridiculous / lost completely different / wrong / aware / overwhelmed strongly opposed / recommended / suggest / believe

---

## 动词 + 副词

think clearly / speak fluently / sleep soundly work closely / listen carefully / fail miserably succeed brilliantly / apologize sincerely / breathe deeply

**记忆策略：** 不要背单个词，永远以**块**为单位记—— 记 `make progress`，不要分别记 `make` 和 `progress`。
