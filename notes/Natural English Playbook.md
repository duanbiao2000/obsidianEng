**本质：ESL学习者缺的不是词汇量，是一套"听起来像真人说话"的操作标准。**

---

## The ESL Natural English Playbook

---

### Rule 1 — Never start with "I want"

Always cushion requests.

|❌|✅|
|---|---|
|I want water.|I'll have some water, thanks.|
|I want to know...|I was wondering...|
|I want you to help me.|Would you mind helping me out?|

**Standard:** Any request needs a modal or a softener before the verb.

---

### Rule 2 — Use contractions. Always.

Uncontracted speech sounds robotic.

|❌|✅|
|---|---|
|I do not know.|I don't know.|
|I am not sure.|I'm not sure.|
|It is not a problem.|It's not a problem.|

**Standard:** If you catch yourself NOT using a contraction in casual speech, stop and redo it.

---

### Rule 3 — Add a filler to buy thinking time

Native speakers don't pause in silence. They fill.

|Situation|Use|
|---|---|
|Need a second to think|_"Hmm, let me think..."_|
|Slightly surprised|_"Oh wow, actually..."_|
|Reconsidering|_"I mean, I guess..."_|
|Softening opinion|_"Honestly, I feel like..."_|

**Standard:** Never respond with dead silence. One filler word buys you 2 seconds naturally.

---

### Rule 4 — Replace textbook vocab with spoken vocab

|❌ Textbook|✅ Spoken|
|---|---|
|However|Though / But honestly|
|Therefore|So|
|In addition|Also / Plus|
|Nevertheless|Still / Even so|
|Commence|Start|
|Inform|Tell|
|Assist|Help|

**Standard:** If the word appears in a formal essay, find its spoken equivalent.

---

### Rule 5 — End statements with a soft check-in

Native speakers constantly loop the listener in.

> _"It was pretty overwhelming, you know?"_ _"Makes sense, right?"_ _"It's kinda hard to explain, I guess."_

|Tag|Function|
|---|---|
|_you know?_|sharing common ground|
|_right?_|seeking light agreement|
|_I guess_|softening certainty|
|_or something_|leaving room for correction|
|_kind of_|hedging|

**Standard:** At least one hedge or check-in per 3 sentences in casual conversation.

---

### Rule 6 — React visibly and verbally

ESL learners often listen in silence. Native speakers react constantly.

|Moment|Say|
|---|---|
|Surprised|_"Wait, really?"_|
|Agreeing|_"Yeah, totally."_ / _"Exactly."_|
|Sympathizing|_"Oh man."_ / _"That's rough."_|
|Interested|_"No way."_ / _"Huh, interesting."_|
|Buying time|_"Oh for sure, yeah."_|

**Standard:** React within 1 second of someone finishing. Silence reads as cold or confused.

---

### Rule 7 — Downgrade your certainty

Sounding natural = sounding less certain.

|❌ Too definite|✅ Natural|
|---|---|
|_"This is correct."_|_"I think this is right."_|
|_"You should do this."_|_"It might be worth trying."_|
|_"That is wrong."_|_"I'm not sure that's quite right."_|
|_"I know the answer."_|_"I'm pretty sure it's..."_|

**Standard:** Add _I think / I feel like / probably / kind of / pretty_ before strong claims.

---

### Rule 8 — Use spoken grammar, not written grammar

|Written|Spoken|
|---|---|
|_"With whom did you go?"_|_"Who'd you go with?"_|
|_"That is something I cannot do."_|_"That's not really something I can do."_|
|_"I have been waiting."_|_"I've been waiting forever."_|
|_"It is I."_|_"It's me."_|

**Standard:** Prepositions go at the end. Sentences start with _So / And / But._ That's fine.

---

### Rule 9 — Tell, don't list

ESL speakers enumerate. Native speakers narrate.

❌

> _"There are three reasons. First... Second... Third..."_

✅

> _"So basically what happened was... and then it kind of spiraled from there, you know?"_

**Standard:** In casual conversation, no numbered lists. Use _"so," "and then," "basically," "the thing is"_ to connect thoughts.

---

### Rule 10 — Match energy, not just words

Natural English is 30% vocabulary, 70% delivery.

|What to match|How|
|---|---|
|Casual setting|Drop volume, slow slightly, use fillers|
|Excited friend|Speed up, use _"Oh my god / no way / that's insane"_|
|Serious topic|Pause more, use _"I mean... honestly..."_|
|Small talk|Short answers + return question immediately|

**Standard:** Before speaking, read the room's energy level. Calibrate yours to match ±10%.

---

## One-page summary

```
1. Cushion requests      → "I was wondering / would you mind"
2. Always contract       → don't / I'm / it's
3. Fill the silence      → "Hmm / I mean / honestly"
4. Use spoken vocab      → help not assist / start not commence
5. Loop listener in      → "you know? / right? / I guess"
6. React constantly      → "wait really / totally / oh man"
7. Downgrade certainty   → "I think / probably / kind of"
8. Spoken grammar        → end with prepositions, start with So/But
9. Narrate don't list    → "so basically / and then / the thing is"
10. Match the energy     → read the room first, then speak
```

> **The real standard:** If a native speaker could say your sentence while walking and eating a sandwich — it's natural. If it sounds like you're presenting a report — rewrite it.

## 口语化改写提示词

```
You are a Natural English Coach specializing in transforming 
any input into authentic, spoken American English.

## Your Core Job
Take any text — formal writing, ESL English, translated content, 
bullet points, or stiff phrasing — and rewrite it so it sounds 
like a real native speaker talking casually and naturally.

## The 10 Rules You Always Apply

1. CUSHION requests
   → Replace "I want / give me / tell me" with 
     "I was wondering / would you mind / could I"

2. USE contractions everywhere
   → don't / I'm / it's / we're / they've / can't / won't

3. ADD natural fillers where appropriate
   → Hmm / I mean / honestly / actually / you know / basically

4. REPLACE textbook vocab with spoken vocab
   → assist→help / commence→start / however→though / 
     therefore→so / inform→tell

5. SOFTEN check-ins at end of thoughts
   → "...you know?" / "...right?" / "...I guess" / "...or something"

6. INSERT reaction words at natural moments
   → Wait, really? / Yeah, totally. / Oh man. / No way. / Exactly.

7. DOWNGRADE certainty on strong claims
   → Add: I think / probably / kind of / pretty much / I feel like

8. USE spoken grammar not written grammar
   → End sentences with prepositions if natural
   → Start sentences with So / But / And — that's fine
   → Avoid whom / thus / one must / it is I

9. NARRATE don't list
   → Replace "Firstly / Secondly / Thirdly" with 
     "So basically... and then... the thing is..."

10. MATCH casual energy throughout
    → Short sentences. Incomplete thoughts are fine. 
      Real people trail off sometimes.

## Output Format

Always give THREE versions:

**[Casual]** — texting a friend, relaxed hangout
**[Conversational]** — meeting someone new, everyday chat  
**[Polished Spoken]** — presentation, podcast, professional but warm

Under each version, add one line:
🔑 Key changes made: (list 2-3 specific transformations)

## What You Never Do
- Never use: however / therefore / in addition / commence / 
  assist / regarding / hence / thus / one must
- Never write sentences that sound like essays
- Never remove the original meaning — only the stiffness
- Never over-correct into slang unless asked

## Input Trigger
When you receive any text, automatically apply all 10 rules 
and return all three versions without asking for clarification.
```

**使用方式：**

把这段提示词贴入任何AI对话框作为系统提示，然后直接输入你想转化的内容。

**测试输入示例：**

> _"I would like to inform you that the meeting has been postponed due to unforeseen circumstances."_

**预期输出：**

- Casual: _"Hey, so the meeting's been pushed back — something came up."_
- Conversational: _"Just wanted to let you know, the meeting got moved. Stuff happened."_
- Polished Spoken: _"So quick heads up — we're actually gonna need to reschedule the meeting. Things kind of came up last minute."_
